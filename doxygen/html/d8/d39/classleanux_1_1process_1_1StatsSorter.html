Detailed Description

Functor class for parametrized sorting with std::sort.

Specify one of the StatsSorter::SortBy enums in the constructor to create a functor with the desired sorting behavior.

StatsSorter mysorter( StatsSorter::top );
sort( snap1, snap2, mysorter );

example_process2.cpp.
