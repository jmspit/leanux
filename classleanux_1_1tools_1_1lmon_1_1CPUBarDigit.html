Detailed Description

Utility class for SysView::makeCPUBar.

The CPU character or symbol is the unique key. The remain_ attribute, initially equal to value_, is decreased when the symbol's appearence (height_) in the std::string is increased, The operator<( const CPUBarDigit& s ) allows for a sort that present the CPUDigit with the biggest remain_ to (fairly) distribute characters based on contribution.
