bool leanux::sysdevice::LoopbackDevice::accept ( SysDevicePath &  path)
virtual

Return true if the SysDevice recognizes itself in the trailing part of path.

If so, the path is stripped so that it ends with the parent device, which may be empty for top level devices. Should be implemented by descendent classes.

Parameters
paththe SysDevicePath to accept.
Returns
true when the SysDevice descendent accepts the path as valid.
