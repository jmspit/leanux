◆ deltaStats()

bool leanux::cpu::deltaStats ( const CPUStatsMap &  earlier,
const CPUStatsMap &  later,
CPUStatsMap &  delta 
)

Compute the deltas for two CPUStatMap std::maps into delta.

There are conditions that could lead to an invalid delta

  • (at least) one of the numbers overflows.
  • the number of CPUs is changed between the stats asked to delta. In both cases, the function returns false indicating the results are invalid.
    Parameters
    earlierthe earlier set of stats.
    laterthe later set of stats.
    deltathe std::map that will hold the resulting delta.
    Returns
    false when a delta would show incorrect data

example_cpu.cpp, and example_process2.cpp.
