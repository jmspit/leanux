◆ getDiskId()

std::string leanux::block::MajorMinor::getDiskId ( ) const

Retrieve a string identifying a disk on a best-efffort basis.

If the disk has a WWN, that is returned, else if the disk has a serial number, that is returned, else the device name is returned, in absence of required truth. virtio and vmware disks can lack a WWN and a serial number, so these disk types are impossible to identify (without tagging their contents).

Returns
a best-effort id for the disk.
